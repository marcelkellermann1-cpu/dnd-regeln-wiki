# Sprachen (Languages)

> Welche Sprachen dein Charakter spricht und versteht.

## Überblick

Jeder Charakter beherrscht **Gemeinsprache (Common)** und zusätzliche Sprachen, die durch Spezies, Hintergrund und Klasse bestimmt werden.

## Standardsprachen

- **Gemeinsprache (Common):** Die Lingua Franca der Zivilisation.

## Spezies-Sprachen

Jede Spezies gewährt zusätzliche Sprachen:

| Spezies | Sprachen |
|---|---|
| Mensch | Gemeinsprache + 1 Wahl |
| Elf | Gemeinsprache, Elfisch |
| Zwerg | Gemeinsprache, Zwergisch |
| Drachenblütiger | Gemeinsprache, Drakonisch |
| Tiefling | Gemeinsprache, Infernalisch |
| Halbling | Gemeinsprache, Halblingisch |
| Gnome | Gemeinsprache, Gnomisch |
| Goliath | Gemeinsprache, Riesisch |
| Ork | Gemeinsprache, Orkisch |
| Aasimar | Gemeinsprache, Celestisch |

## Optionale Sprachen

Folgende Sprachen stehen üblicherweise zur Wahl:

- **Standardspezies-Sprachen:** Zwergisch, Elfisch, Gnomisch, Halblingisch, Orkisch
- **Drakonisch** (Drachen)
- **Tiefen-Sprech (Undercommon)** (Unterwelt)
- **Riesisch** (Riesen, Oger)
- **Infernalisch** (Teufel)
- **Celestisch** (Celestials)
- **Abyssisch** (Dämonen)
- **Druidisch** (geheime Sprache der Druiden)
- **Diebessprache (Thieves' Cant)** (Verbrecher)
- **Primordial** (Elementare: Aquan, Ignan, Terran, Avian)
- **Weitere** je nach Setting

## Klassen-Sprachen

- **Barde** erhält 2 zusätzliche Sprachen ab Stufe 1.
