# Persönlichkeit – Wesenszüge (Personality Traits)

> Zwei kurze Wesenszüge beschreiben, wie dein Charakter wirkt, spricht und sich verhält.

## Überblick

Wesenszüge sind kurze Beschreibungen, die zeigen, wie dein Charakter mit anderen interagiert. Sie helfen dir, deinen Charakter am Spieltisch zum Leben zu erwecken, und geben dem Spielleiter Anregungen für deine Rolle im Spiel.

Dein Hintergrund schlägt dir Wesenszüge vor (siehe die Tabelle deines Hintergrunds in Kapitel 4). Du kannst diese Vorschläge übernehmen, sie anpassen oder eigene Wesenszüge formulieren.

## Beispiele

| Wesenszug |
|---|
| Ich stottere, wenn ich nervös bin. |
| Ich gebe nie auf, wenn ich eine Herausforderung sehe. |
| Ich sammle ungewöhnliche Gegenstände, um mein Studierzimmer zu füllen. |
| Ich bin immer höflich und respektvoll. |
| Ich kann die Beweggründe anderer leicht durchschauen. |
| Ich bin gegenüber Autoritätspersonen misstrauisch. |
| Ich mag es, im Mittelpunkt zu stehen. |
| Ich habe einen scharfen Witz für jede Situation. |
| Ich rede mit mir selbst, wenn ich nachdenke. |
| Ich versuche, jedes neue Wort in einer Sprache, die ich nicht kenne, auszusprechen. |
