# Multiclassing

> Optionale Regelung: Charaktere können Stufen in mehreren Klassen haben.

## Überblick

Mit Zustimmung des Spielleiters kann ein Charakter beim Stufenaufstieg eine Stufe in einer neuen Klasse erhalten, anstatt in seiner aktuellen Klasse aufzusteigen.

## Voraussetzungen

Um eine neue Klasse zu multiclassen, muss das Primärattribut (oder ein alternatives) der neuen Klasse **13 oder höher** sein:

| Neue Klasse | Benötigtes Attribut |
|---|---|
| Barbar | Stärke 13+ |
| Barde | Charisma 13+ |
| Druide | Weisheit 13+ |
| Hexenmeister | Charisma 13+ |
| Kämpfer | Stärke **oder** Geschicklichkeit 13+ |
| Kleriker | Weisheit **oder** Charisma 13+ |
| Magier | Intelligenz 13+ |
| Mönch | Geschicklichkeit **oder** Weisheit 13+ |
| Paladin | Stärke **oder** Charisma 13+ |
| Schurke | Geschicklichkeit 13+ |
| Waldläufer | Geschicklichkeit **oder** Weisheit **oder** Charisma 13+ |
| Zauberer | Charisma 13+ |

## Effekte

- **Trefferpunkte:** Du erhältst Trefferpunkte basierend auf dem Trefferpunktwürfel der neuen Klasse + Konstitutionsmodifikator (keine Maximalwurf-Regel der Stufe 1).
- **Übungsbonus:** Entspricht deiner Gesamtstufe, nicht der Klassenstufe.
- **Rettungswürfe:** Du erhältst die primären Rettungswürfe der neuen Klasse (falls nicht bereits vorhanden).
- **Übungen:** Leichte Rüstung, ein Werkzeug oder eine Fertigkeit der neuen Klasse (falls nicht bereits besser).
- **Zauberslots:** Berechnen sich nach der Multiclass-Zauberslot-Tabelle im Spielerhandbuch.
