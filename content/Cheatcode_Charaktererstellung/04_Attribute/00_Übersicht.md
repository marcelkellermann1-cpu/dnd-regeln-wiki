# Attribute – Übersicht

> Die sechs Attribute (engl. *Ability Scores*) bilden die Grundlage jedes Charakters in D&D 5e.

## Die sechs Attribute

| Attribut | Abk. | Hauptlink | Bedeutung |
|---|---|---|---|
| [[Staerke|Stärke]] | STR | → [[../../D&D Spielerhandbuch 2024/Anhang_C_Regelglossar/Weisheit|PHB 2024]] | Körperliche Kraft, Tragfähigkeit, Nahkampf |
| [[Geschicklichkeit|Geschicklichkeit]] | GES | → [[../../D&D Spielerhandbuch 2024/Anhang_C_Regelglossar/Geschicklichkeit|PHB 2024]] | Beweglichkeit, Reflexe, Initiative |
| [[Konstitution|Konstitution]] | KON | → [[../../D&D Spielerhandbuch 2024/Anhang_C_Regelglossar/Konstitution|PHB 2024]] | Ausdauer, Gesundheit, HP pro Stufe |
| [[Intelligenz|Intelligenz]] | INT | → [[../../D&D Spielerhandbuch 2024/Anhang_C_Regelglossar/Intelligenz|PHB 2024]] | Logik, Gedächtnis, Analyse |
| [[Weisheit|Weisheit]] | WEI | → [[../../D&D Spielerhandbuch 2024/Anhang_C_Regelglossar/Weisheit|PHB 2024]] | Wahrnehmung, Intuition, Willenskraft |

## Modifikator-Tabelle

Jeder Attributswert (1–30) hat einen **Modifikator**:

| Wert | Modifikator | | Wert | Modifikator |
|---|---|---|---|---|
| 1 | –5 | | 16–17 | +3 |
| 2–3 | –4 | | 18–19 | +4 |
| 4–5 | –3 | | 20–21 | +5 |
| 6–7 | –2 | | 22–23 | +6 |
| 8–9 | –1 | | 24–25 | +7 |
| 10–11 | +0 | | 26–27 | +8 |
| 12–13 | +1 | | 28–29 | +9 |
| 14–15 | +2 | | 30 | +10 |

Formel: `(Wert − 10) ÷ 2, abgerundet`.

## Methoden zur Attributsbestimmung

Vergleiche und Details siehe [[Methoden|Methoden]].

| Methode | Typisch für | Balance |
|---|---|---|
| Standard-Array (15,14,13,12,10,8) | Einsteiger, ausgeglichen | sehr ausgeglichen |
| 4W6 drop lowest (6×) | Erfahrene Spieler | variabel, oft besser |
| Punkteverteilung (27 Punkte) | Kontroll-Freaks | vollständig kontrolliert |
| Manuell (DM-Erlaubnis) | Kampagnen mit festen Werten | DM-abhängig |

## Klassen und Primärattribute

Jede Klasse hat ein **Primärattribut** (Klasse-Tabelle):

| Attribut | Klassen (Primär) |
|---|---|
| **STR** | Barbar |
| **GES** | Schurke (alternative), Kämpfer (alternative) |
| **KON** | – (universell wichtig für HP) |
| **INT** | Magier |
| **WEI** | Druide, Kleriker (alternative), Waldläufer (alternative), Mönch (alternative) |
| **CHA** | Barde, Hexenmeister, Paladin (alternative), Zauberer, Kleriker (alternative), Waldläufer (alternative), Mönch (alternative) |

## Wichtige Begriffe

- **Attributswert:** Zahl von 1 bis 30
- **Modifikator:** Abgeleiteter Bonus/Malus (Tabelle oben)
- **Übung (Proficiency):** Wird zusätzlich addiert, wenn das Attribut „trainiert" ist (z. B. Rettungswurf)
- **Speicherbonus:** Hintergrund/Spezies geben +1/+2/+3 auf einzelne Attribute
