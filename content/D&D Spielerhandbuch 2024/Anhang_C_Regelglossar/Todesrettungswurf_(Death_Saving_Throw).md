# Todesrettungswurf (Death Saving Throw)

Ein Spielercharakter muss einen Todesrettungswurf (auch Todessave genannt) ablegen, wenn er seinen Zug mit 0 Trefferpunkten beginnt. Siehe auch Kapitel 1 („Schaden und Heilung").
