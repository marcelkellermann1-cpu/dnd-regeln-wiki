# Freundlich [Einstellung] (Friendly)

Eine freundliche Kreatur betrachtet dich wohlwollend. Du hast Vorteil auf einen Fähigkeitswurf, um eine freundliche Kreatur zu beeinflussen. Siehe auch „Einflussnahme".
