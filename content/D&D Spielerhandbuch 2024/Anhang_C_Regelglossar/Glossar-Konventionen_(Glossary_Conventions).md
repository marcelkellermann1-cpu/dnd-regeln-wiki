# Glossar-Konventionen und Abkürzungen (Glossary Conventions)

Das Glossar verwendet die folgenden Konventionen.

**Tags in Klammern.** Einige Einträge besitzen ein Tag in Klammern nach dem Namen des Eintrags, wie etwa „Angriff [Aktion]". Ein Tag – Aktion, Wirkungsbereich, Einstellung, Zustand oder Gefahr – zeigt an, dass eine Regel Teil einer Regelfamilie ist. Die Tags besitzen ebenfalls Glossareinträge.

**„Du".** Die Regeln des Spiels – in diesem Glossar und anderswo – sprechen oft davon, dass etwas mit „dir" in der Spielwelt geschieht. Dieses „Du" bezieht sich auf die Kreatur oder das Objekt, auf das die Regel in einem bestimmten Spielmoment zutrifft. Zum Beispiel bezieht sich das „Du" im Zustand Liegend auf eine Kreatur, die diesen Zustand gerade besitzt.

**„Siehe auch".** Einige Glossareinträge enthalten einen Abschnitt „Siehe auch", der auf andere Einträge im Glossar, auf Kapitel in diesem Buch oder auf beides verweist.

**Keine veralteten Begriffe.** Das Glossar enthält nur Definitionen aktueller Regelbegriffe. Wenn du einen Begriff aus einer früheren Version der fünften Edition suchst, schlage im Index nach.

**Abkürzungen.** Die unten aufgeführten Abkürzungen erscheinen in diesem Glossar und an anderen Stellen in den Regeln.
