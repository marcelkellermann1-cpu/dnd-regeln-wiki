# Schwerpunktkenntnis (Proficiency)

Wenn du Schwerpunktkenntnis für etwas besitzt, kannst du deinen Schwerpunktbonus auf jede W20-Probe addieren, die du unter Verwendung dieses Dings durchführst. Eine Kreatur kann Schwerpunktkenntnis in einer Fertigkeit, einem Rettungswurf, einer Waffe oder einem Werkzeug haben. Siehe auch Kapitel 1 („Schwerpunkt").
