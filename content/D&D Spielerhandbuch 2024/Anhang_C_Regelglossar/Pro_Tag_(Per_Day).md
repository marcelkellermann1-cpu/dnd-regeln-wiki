# Pro Tag (Per Day)

Wenn eine Regel besagt, dass du etwas eine bestimmte Anzahl von Malen pro Tag verwenden kannst, bedeutet das, dass du eine Lange Rast beenden musst, um es erneut zu verwenden, nachdem du die Anwendungen aufgebraucht hast.
