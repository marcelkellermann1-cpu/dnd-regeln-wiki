# Feindselig [Einstellung] (Hostile)

Eine feindselige Kreatur betrachtet dich ablehnend. Du hast Nachteil auf einen Fähigkeitswurf, um eine feindselige Kreatur zu beeinflussen. Siehe auch „Einflussnahme".
