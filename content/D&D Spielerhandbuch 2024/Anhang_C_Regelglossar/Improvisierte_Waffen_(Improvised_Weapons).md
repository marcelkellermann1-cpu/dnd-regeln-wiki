# Improvisierte Waffen (Improvised Weapons)

Eine improvisierte Waffe ist ein Objekt, das als behelfsmäßige Waffe verwendet wird, wie zerbrochenes Glas, ein Tischbein oder eine Bratpfanne. Eine Einfache oder Martialische Waffe zählt ebenfalls als improvisierte Waffe, wenn sie entgegen ihrer Bestimmung verwendet wird; wenn du eine Fernkampfwaffe verwendest, um einen Nahkampfangriff auszuführen, oder eine Nahkampfwaffe ohne die Wurf-Eigenschaft wirfst, zählt die Waffe als improvisierte Waffe. Eine improvisierte Waffe folgt den folgenden Regeln.

**Schwerpunkt.** Addiere deinen Schwerpunktbonus nicht auf Angriffswürfe mit einer improvisierten Waffe.

**Schaden.** Bei einem Treffer fügt die Waffe 1W4 Schaden eines Typs zu, den der SL für das Objekt als angemessen erachtet.

**Reichweite.** Wenn du die Waffe wirfst, hat sie eine normale Reichweite von 20 Fuß und eine lange Reichweite von 60 Fuß.

**Waffenäquivalente.** Wenn eine improvisierte Waffe einer Einfachen oder Martialischen Waffe ähnelt, kann der SL sagen, dass sie wie diese Waffe funktioniert und deren Regeln verwendet. Beispielsweise könnte der SL ein Tischbein als Knüppel behandeln.
