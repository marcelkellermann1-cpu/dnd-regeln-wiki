# Schadenstypen (Damage Types)

Angriffe und andere schädliche Effekte fügen unterschiedliche Schadenstypen zu. Schadenstypen besitzen keine eigenen Regeln, aber andere Regeln, wie Widerstandsfähigkeit, stützen sich auf diese Typen. Die Tabelle „Schadenstypen" bietet Beispiele, um dem SL zu helfen, einem neuen Effekt einen Typ zuzuordnen.

### Schadenstypen

| Typ | Beispiele |
|---|---|
| Säure (Acid) | Ätzende Flüssigkeiten, Verdauungsenzyme |
| Wuchtig (Bludgeoning) | Stumpfe Gegenstände, Umschnürung, Sturz |
| Kälte (Cold) | Gefrierendes Wasser, eisige Schläge |
| Feuer (Fire) | Flammen, unerträgliche Hitze |
| Kraft (Force) | Reine magische Energie |
| Blitz (Lightning) | Elektrizität |
| Nekrotisch (Necrotic) | Lebensentziehende Energie |
| Stich (Piercing) | Klauen, stechende Gegenstände |
| Gift (Poison) | Giftiges Gas, Venom |
| Psychisch (Psychic) | Geistsprengende Energie |
| Strahlend (Radiant) | Heilige Energie, sengende Strahlung |
| Hieb (Slashing) | Klauen, schneidende Gegenstände |
| Donner (Thunder) | Druckwellen-artiger Schall |
