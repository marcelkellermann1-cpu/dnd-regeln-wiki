# Kreaturentyp (Creature Type)

Jede Kreatur, einschließlich jedes Spielercharakters, besitzt ein Tag in den Regeln, das den Typ der Kreatur kennzeichnet. Die meisten Spielercharaktere sind vom Typus Humanoid. Dies sind die Kreaturentypen des Spiels:

| | | | |
|---|---|---|---|
| Aberration | Elementar | Monstruosität | Bestie (Beast) |
| Feenwesen | Schlamm (Ooze) | Himmlisch (Celestial) | Teufel (Fiend) |
| Pflanze (Plant) | Konstrukt (Construct) | Riese (Giant) | Untot (Undead) |
| Drache (Dragon) | Humanoid | | |

Die Typen besitzen keine eigenen Regeln, aber einige Regeln im Spiel beeinflussen Kreaturen bestimmter Typen auf unterschiedliche Weise.
