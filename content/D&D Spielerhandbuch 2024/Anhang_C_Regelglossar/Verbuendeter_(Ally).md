# Verbündeter (Ally)

Eine Kreatur ist dein Verbündeter, wenn sie Mitglied deiner Abenteurergruppe ist, dein Freund, auf deiner Seite im Kampf steht oder eine Kreatur, die durch die Regeln oder den SL als dein Verbündeter bestimmt wird.
