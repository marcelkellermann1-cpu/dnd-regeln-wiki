# Schadensresistenz (Resistance)

Wenn du Widerstand gegen einen Schadenstyp hast, wird Schaden dieses Typs gegen dich halbiert (abgerundet). Widerstand wird auf eine Schadensinstanz nur einmal angewendet. Siehe auch Kapitel 1 („Schaden und Heilung").
