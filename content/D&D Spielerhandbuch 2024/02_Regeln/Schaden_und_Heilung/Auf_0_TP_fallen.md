# Auf 0 Trefferpunkte fallen

Wenn eine Kreatur auf 0 Trefferpunkte fällt, stirbt sie entweder sofort oder wird bewusstlos, wie unten erklärt.

## Sofortiger Tod

Hier sind die Hauptwege, wie eine Kreatur sofort sterben kann:

### Monstertod
Ein Monster **stirbt in dem Moment**, in dem es auf 0 Trefferpunkte fällt. Ein Spielleiter kann diese Regel für ein einzelnes Monster ignorieren und es wie einen Charakter behandeln.

### Trefferpunktemaximum von 0
Eine Kreatur **stirbt**, wenn ihr Trefferpunktemaximum 0 erreicht. Bestimmte Effekte entziehen Lebensenergie und reduzieren das Trefferpunktemaximum einer Kreatur.

### Massiver Schaden
Wenn Schaden einen Charakter auf 0 Trefferpunkte reduziert und **Schaden übrig bleibt**, stirbt der Charakter, wenn der Rest sein Trefferpunktemaximum erreicht oder überschreitet.

#### Beispiel
Wenn dein Charakter ein Trefferpunktemaximum von 12 hat, aktuell 6 Trefferpunkte hat und 18 Schaden erleidet:
- Der Charakter fällt auf 0 Trefferpunkte
- Es bleiben 12 Schaden übrig
- Der Charakter **stirbt**, da 12 seinem Trefferpunktemaximum entspricht

## Charaktertod

Wenn dein Charakter stirbt, könnten andere einen magischen Weg finden, ihn wiederzubeleben, wie mit dem Zauber *Totenerweckung* (Raise Dead).

Oder sprich mit dem SL darüber, einen neuen Charakter zu erstellen, der sich der Gruppe anschließt. Das Regelglossar enthält weitere Informationen darüber, tot zu sein.
