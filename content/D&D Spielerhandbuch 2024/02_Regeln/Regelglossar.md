# Regelglossar (PHB 2024)

> Vereinfachte und überarbeitete Regeln aus Anhang C des Spielerhandbuchs 2024.

## Schadenstypen (Damage Types)
Verschiedene Angriffe, schadensverursachende Zauber und andere schädliche Effekte verursachen unterschiedliche Schadenstypen. Andere Regeln (z. B. Resistenz, Verwundbarkeit) hängen von diesen Typen ab.

| Typ | Beispiel |
|---|---|
| Säure (Acid) | Schwarzer-Drachen-Atem, Schwarzer-Pudding-Enzyme |
| Wucht (Bludgeoning) | Hämmer, Fallen, Einschnürung |
| Kälte (Cold) | Eisteufel-Lanze, Weißer-Drachen-Atem |
| Feuer (Fire) | Roter-Drachen-Atem, viele Zauber |
| Energie (Force) | Magisches Geschoss, Geistige Waffe |
| Blitz (Lightning) | Blitzschlag, Blauer-Drachen-Atem |
| Nekrotisch (Necrotic) | Untote-Fähigkeiten, Gruselige Berührung |
| Stich (Piercing) | Speere, Monsterbisse |
| Gift (Poison) | Grüner-Drachen-Atem, Stiche |
| Psychisch (Psychic) | Gedankenschinder-Psionic |
| Strahlend (Radiant) | Feuersturm, Engels-Waffe |
| Hieb (Slashing) | Schwerter, Äxte, Klauen |
| Donner (Thunder) | Donnerschlag-Zauber |

- Geblendet, Bezaubert, Taub, Erschöpft, Erschreckt, Gefesselt, Eingeschränkt, Unsichtbar, Gelähmt, Versteinert, Vergiftet, Liegend, Betäubt, Bewusstlos, Verwirrt

## Wichtige Spielregeln (PHB 2024)

### Vorteil und Nachteil
Würfle einen zweiten W20 und nimm den höheren (Vorteil) oder niedrigeren (Nachteil) Wert. Mehrere Vorteile/Nachteile kombinieren sich nicht. Wenn beides gilt, heben sie sich auf.

### W20-Tests
Fähigkeitswürfe, Angriffswürfe und Rettungswürfe sind die drei Haupttypen von W20-Würfen. Alle folgen diesen Schritten:
1. Würfle den W20 und addiere einen Modifikator.
2. Wende situationsbedingte Boni und Mali an.
3. Vergleiche das Ergebnis mit dem Zielwert (SG für Würfe/Rettungswürfe, RK für Angriffe).

### Übungsbonus
Bestimmt durch Charakterstufe (nicht Klasse): +2 (Stufe 1-4), +3 (5-8), +4 (9-12), +5 (13-16), +6 (17-20).

### Fähigkeitswürfe vs. Rettungswürfe
- **Fähigkeitswurf:** Reaktion auf eine unsichere Situation.
- **Rettungswurf:** Reaktion auf einen bestimmten Effekt (Zauber, Falle, Bedrohung).

## Dauerhafte Trefferpunkte
Temporäre Trefferpunkte sind kein Ersatz für aktuelle Trefferpunkte. Bei mehreren Quellen: Du entscheidest, welche du behältst.
