# Wort des Strahlens (Word of Radiance)

## Eigenschaften
| | |
|---|---|
| **Grad** | Cantrip |
| **Schule** | Beschwörung |
| **Wirklzeit** | 1 Aktion |
| **Reichweite** | Selbst (1,5 m Radius) |
| **Komponenten** | V, M (heiliges Symbol) |
| **Dauer** | Sofort |
| **Klassen** | [[Kleriker]] |

## Beschreibung
Du sprichst ein göttliches Wort und strahlende Energie bricht aus dir hervor. Alle Kreaturen deiner Wahl in 1,5 m müssen einen Konstitutionsrettungswurf ablegen.

- **Bei Misserfolg:** 1W6 Strahlungsschaden

## Skalierung
| Charakterstufe | Schaden |
|----------------|---------|
| 1–4 | 1W6 |
| 5–10 | 2W6 |
| 11–16 | 3W6 |
| 17–20 | 4W6 |

## Taktische Hinweise
- Trifft mehrere Ziele gleichzeitig
- Du wählst welche Kreaturen betroffen sind (Verbündete ausgenommen)
- Ideal wenn du von Feinden umzingelt bist
- Schwächer als Sacred Flame gegen einzelne Ziele
