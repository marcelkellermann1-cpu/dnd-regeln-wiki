# Naturwächter (Guardian of Nature) – Grad 4

## Eigenschaften
| | |
|---|---|
| **Grad** | 4 |
| **Schule** | Transmutation |
| **Wirkzeit** | Bonusaktion |
| **Reichweite** | Selbst |
| **Komponenten** | V |
| **Dauer** | Konzentration, bis zu 1 Minute |
| **Klassen** | [[Druide]], [[Waldläufer]] |

## Beschreibung
Du nimmst eine mächtige Naturform an. Wähle eine von zwei Formen:

### Urwüchsiges Tier
- Dunkelsicht 36 m
- Vorteil auf Stärkewürfe
- Nahkampfwaffen gelten als magisch
- Vorteil auf Konstitutionsrettungswürfe (Konzentration halten)

### Großer Baum
- Dunkelsicht 36 m
- Vorteil auf Weisheitswürfe
- Vorteil auf Fernkampf-Angriffswürfe
- Ranken erschweren Bewegung in 3 m (schwieriges Gelände)

## Taktische Hinweise
- Urwüchsiges Tier: Nahkampf-Boost
- Großer Baum: Fernkampf + Kontrolle
- Nur verbale Komponente
- Dunkelsicht auf beiden Formen sehr nützlich
