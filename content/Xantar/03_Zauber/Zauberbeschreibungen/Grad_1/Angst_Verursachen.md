# Angst verursachen (Cause Fear)

## Eigenschaften
| | |
|---|---|
| **Grad** | 1 |
| **Schule** | Nekromantie |
| **Wirkzeit** | 1 Aktion |
| **Reichweite** | 18 m |
| **Komponenten** | V |
| **Dauer** | Konzentration, bis zu 1 Minute |
| **Klassen** | [[Hexenmeister]], [[Paladin]] |

## Beschreibung
Wähle eine Kreatur in Reichweite die du sehen kannst. Sie muss einen Weisheitsrettungswurf ablegen.

- **Bei Misserfolg:** Kreatur ist erschreckt vor dir für die Dauer
- Am Ende jedes ihrer Züge: erneuter Rettungswurf, bei Erfolg endet der Zauber
- Wirkt nicht auf Untote oder Konstrukte

## Höhere Grade
- Ab Grad 2: Für jeden zusätzlichen Grad kannst du eine weitere Kreatur als Ziel wählen

## Taktische Hinweise
- Erschreckt = Nachteil auf Angriffe + keine freiwillige Bewegung auf dich zu
- Konzentrationsbasiert – gut für anhaltende Kontrolle
- Einzelziel auf Grad 1, skaliert gut
- Nicht gegen Untote nutzbar (relevant für Nekromanten-Kampagnen)
