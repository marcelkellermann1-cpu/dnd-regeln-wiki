# Elemente absorbieren (Absorb Elements)

## Eigenschaften
| | |
|---|---|
| **Grad** | 1 |
| **Schule** | Abjuration |
| **Wirkzeit** | Reaktion (wenn du Säure, Kälte, Feuer, Blitz oder Donner nimmst) |
| **Reichweite** | Selbst |
| **Komponenten** | G |
| **Dauer** | 1 Runde |
| **Klassen** | [[Druide]], [[Magier]], [[Waldläufer]], [[Zauberer]] |

## Beschreibung
Als Reaktion wenn du Säure-, Kälte-, Feuer-, Blitz- oder Donnerschaden nimmst:
- Du erhältst **Resistenz** gegen diesen Schadenstyp für diesen Angriff
- Dein nächster Nahkampfangriff bis Ende deines nächsten Zuges verursacht **+1W6** dieses Schadenstyps

## Taktische Hinweise
- Sehr effizient: Schaden reduzieren UND Gegenangriff
- Keine Konzentration nötig
- Einzige Komponente: Gestik (kein Sprechen nötig)
- Besonders gut gegen Drachen und Elementare
