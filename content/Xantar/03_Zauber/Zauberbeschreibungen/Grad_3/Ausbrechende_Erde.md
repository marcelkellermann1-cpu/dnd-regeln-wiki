# Ausbrechende Erde (Erupting Earth)

## Eigenschaften
| | |
|---|---|
| **Grad** | 3 |
| **Schule** | Transmutation |
| **Wirkzeit** | 1 Aktion |
| **Reichweite** | 36 m |
| **Komponenten** | V, G, M (Schmutzklumpen) |
| **Dauer** | Sofort |
| **Klassen** | [[Druide]], [[Magier]], [[Waldläufer]], [[Zauberer]] |

## Beschreibung
Du lässt den Boden in einem 4,5 m Würfel explodieren. Alle Kreaturen im Bereich: Geschicklichkeitsrettungswurf.

- **Bei Misserfolg:** 3W12 Stumpfschaden
- **Bei Erfolg:** Halber Schaden
- Bereich wird schwieriges Gelände (bleibt dauerhaft)

| usw. | +1W12 |

## Taktische Hinweise
- Hoher Schaden mit W12 Würfeln
- Dauerhaftes schwieriges Gelände als strategischer Bonus
- Nur auf natürlichem Boden möglich
- Synergie mit anderen Flächenzaubern
