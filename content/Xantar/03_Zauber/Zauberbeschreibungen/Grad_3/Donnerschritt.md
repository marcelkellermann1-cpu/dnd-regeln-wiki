# Donnerschritt (Thunder Step) – Grad 3

## Eigenschaften
| | |
|---|---|
| **Grad** | 3 |
| **Schule** | Beschwörung |
| **Wirkzeit** | 1 Aktion |

| **Dauer** | Sofort |
| **Klassen** | [[Magier]], [[Zauberer]] |

## Beschreibung
Du teleportierst dich an einen freien Ort in Reichweite. Ein Donnerschlag entsteht am Ausgangsort.

- Alle Kreaturen in 1,5 m des Ausgangsortes: Konstitutionsrettungswurf
- **Bei Misserfolg:** 3W10 Donnerschaden
- **Bei Erfolg:** Halber Schaden
- Du kannst eine Kreatur mitnehmen die du hältst oder die freiwillig mitkommt (selbe Größe oder kleiner)

## Taktische Hinweise
- Teleportation + Schaden = sehr effizient
- Nur verbale Komponente – in Fesselung möglich
- Kreatur mitnehmen = gute Rettungsoption für Verbündete
- Synergie mit Sturmzauberer
