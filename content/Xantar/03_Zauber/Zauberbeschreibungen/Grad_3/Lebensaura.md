# Lebensaura (Aura of Vitality) – Grad 3

## Eigenschaften
| | |
|---|---|
| **Grad** | 3 |
| **Schule** | Beschwörung |
| **Wirkzeit** | 1 Aktion |
| **Reichweite** | Selbst (9 m Radius) |
| **Komponenten** | V |
| **Dauer** | Konzentration, bis zu 1 Minute |
| **Klassen** | [[Kleriker]], [[Paladin]] |

## Beschreibung
Heilende Energie strahlt von dir aus.

- Als Bonusaktion kannst du jeden Zug eine Kreatur in 9 m heilen
- Heilung: 2W6 TP pro Bonusaktion
- Über 1 Minute (10 Runden): bis zu 20W6 Heilung mit einem Grad-3-Platz

## Taktische Hinweise
- Effizienteste Heilung pro Zauberplatz im Spiel
- Bonusaktion = keine Aktionen verschwendet
- Konzentration nötig – auf Schaden verzichten
- Synergie mit Paladinen die keine anderen Konzentrationszauber benötigen
