# Feinde überall (Enemies Abound) – Grad 3

## Eigenschaften
| | |
|---|---|
| **Grad** | 3 |
| **Schule** | Enchantment |
| **Wirkzeit** | 1 Aktion |
| **Reichweite** | 36 m |

| **Dauer** | Konzentration, bis zu 1 Minute |
| **Klassen** | [[Barde]], [[Hexenmeister]], [[Magier]], [[Zauberer]] |

## Beschreibung
Du greifst in den Geist einer Kreatur ein und verwandelst alle umliegenden Wesen in Feinde.

- Intelligenzrettungswurf gegen deinen Zauber-SG
- **Bei Misserfolg:** Kreatur kann nicht unterscheiden wer Freund oder Feind ist
- Sie greift zufällig eine Kreatur in Reichweite an (am Ende jedes Zuges neu bestimmt)
- Am Ende jedes Zuges: erneuter Rettungswurf möglich

## Taktische Hinweise
- Extrem stark gegen mächtige Einzelziele
- Lässt Bosse die eigene Gruppe angreifen
- Intelligenzrettungswurf – gut gegen niedrige Intelligenz (Bestien, Konstrukte meist immun)
- Konzentration nötig
