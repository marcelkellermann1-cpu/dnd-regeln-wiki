# Gedankendorn (Mind Spike)

## Eigenschaften
| | |
|---|---|
| **Grad** | 2 |
| **Schule** | Divination |
| **Wirkzeit** | 1 Aktion |
| **Reichweite** | 18 m |
| **Komponenten** | G |
| **Dauer** | Konzentration, bis zu 1 Stunde |
| **Klassen** | [[Hexenmeister]], [[Magier]], [[Zauberer]] |

## Beschreibung
Du stechst mit psionischer Energie in den Geist einer Kreatur. Weisheitsrettungswurf.

- **Bei Misserfolg:** 3W8 psychischer Schaden + du kennst den genauen Standort der Kreatur für die Dauer (sofern sie im selben Plan ist)
- **Bei Erfolg:** Halber Schaden, kein Ortungsefffekt

## Höhere Grade
| Grad | Schaden |
|------|---------|
| 2 | 3W8 |
| 3 | 5W8 |
| 4 | 7W8 |
| 5 | 9W8 |

## Taktische Hinweise
- Ortungseffekt überwindet Unsichtbarkeit und magische Verstecken
- Sehr gut gegen fliehende oder unsichtbare Gegner
- Weisheitsrettungswurf – gut gegen niedrige Weisheit
- Konzentration für Ortung nötig
