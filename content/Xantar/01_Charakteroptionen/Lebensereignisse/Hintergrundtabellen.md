# Hintergrundtabellen & Klassenausbildung (S. 64)

Würfle W6 für deinen Hintergrund und deine Klasse um zu erklären wie du dahin kamst.

## Zuhause der Kindheit (W100 + Lebenstil-Mod.)

| Ergebnis | Zuhause |
|----------|---------|
| 0 oder niedriger | Auf der Straße |
| 01–20 | Heruntergekommene Hütte |
| 21–30 | Kein fester Wohnsitz |
| 31–40 | Lager/Dorf in der Wildnis |
| 41–50 | Wohnung in heruntergekommener Gegend |
| 51–70 | Kleines Haus |
| 71–90 | Großes Haus |
| 91–110 | Herrenhaus |
| 111+ | Palast oder Schloss |

## Kindheitserinnerungen (3W6 + Charismamodifikator)

| Ergebnis | Erinnerung |
|----------|-----------|
| 3 oder niedriger | Von anderen Kindern schlecht behandelt |
| 4–5 | Großteil der Kindheit alleine verbracht |
| 6–8 | Für eigenartig gehalten, wenige Freunde |
| 9–12 | Einige enge Freunde, normale Kindheit |
| 13–15 | Mehrere Freunde, überwiegend glückliche Kindheit |
| 16–17 | Leicht Freundschaften geschlossen, viele Freunde |
| 18+ | Bekannt und überall Freunde |
