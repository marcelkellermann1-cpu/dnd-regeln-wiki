# Sassy Sally Jane

> *Die legendäre Puppe von Captain Dapplewing's Manor*

**Sassy Sally Jane** ist eine **Puppe mit rosa Haaren** in einem **patchwork-karierten Kleid**. Sie ist **der Gesprächsstoff des Campus seit Jahrzehnten**.

## Hintergrund

Im Inneren der Puppe befindet sich eine **Sprachbox**, die einige kluge Studenten so konstruiert haben, dass sie immer dann **unhöfliche Sätze wiederholt**, wenn die Hand der Puppe gedrückt wird. **Witzbolde** benutzten sie ausgiebig, um **Fakultätsmitglieder und Angestellte zu ärgern**, bevor sie **beschlagnahmt** wurde.

Seitdem sitzt die Puppe im **Dachgeschoss von Captain Dapplewing's Manor**, neben anderen kaputten, fehlerhaften oder verbotenen Gegenständen.

## Rolle im Abenteuer

Eine geheime Notiz fordert die Charaktere auf, Sassy Sally Jane aus Captain Dapplewing's Manor zu bergen — eine Campus-Mutprobe. Wenn die Charakteren die Puppe erfolgreich bergen, dürfen sie sie behalten. Wenn sie um die Puppe gebeten werden, können sie sie einem Studenten geben, dessen jüngerer Bruder sie bekommen soll, im Austausch gegen einen **Driftglobe**.

> **Achtung:** Offensichtlicher Gebrauch der Puppe in der Nähe von Fakultätsmitgliedern führt dazu, dass Sassy Sally Jane **erneut beschlagnahmt** wird.
